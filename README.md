# Aetherion Cycle

A futuristic immersive web experience built with modern web technologies, advanced UI effects, and cinematic design principles.

## Overview

Aetherion Cycle is a high-concept interactive project focused on creating a next-generation digital experience with:

* Futuristic holographic UI
* Advanced animations and transitions
* Immersive visual effects
* Modern responsive layouts
* High-performance frontend architecture
* Interactive components and dynamic systems

The project combines sci-fi aesthetics with modern web engineering to create a visually rich and highly interactive platform.

## Project Structure

```bash
artifacts/          # Generated assets and outputs

lib/                # Core libraries and utilities
scripts/            # Automation and helper scripts
node_modules/       # Installed dependencies
package.json        # Project configuration
pnpm-lock.yaml      # Dependency lockfile
tsconfig.json       # TypeScript configuration
```

## Installation

Clone the repository:

```bash
git clone https://github.com/krish261108/aetherion-project.git
```

Navigate into the project:

```bash
cd aetherion-project
```

Install dependencies:

```bash
pnpm install
```

---

## Running the Project

Start development server:

```bash
pnpm run dev
```

Or check available scripts:

```bash
pnpm run
```
